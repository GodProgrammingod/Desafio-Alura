Explorando o HTML

Nessa página, você poderá ver os conceitos e interações possíveis com HTML.

1. Estrutura Básica de uma Página HTML

A estrutura básica de uma página HTML pode ser comparada à construção de uma casa. Cada tag tem um papel específico, como se fosse uma parte dessa construção. Vamos entender isso:

  • <!DOCTYPE html>: Pense nisso como a placa na entrada da obra. Ela avisa aos construtores (navegadores) qual o estilo da construção (HTML5).
  • <html>: Esta tag é como as paredes externas da casa. Tudo o que está dentro dela faz parte da estrutura principal do site.
  • <head>: O "sótão" da casa, onde guardamos informações importantes que o usuário não vê diretamente, como o título e os metadados.
  • <body>: Aqui é a sala de estar, onde colocamos o conteúdo visível da página, como textos e imagens.
<!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <h1>Bem-vindo ao HTML!</h1>
    <p>Esse é um exemplo de HTML básico.</p>
  </body>
</html>

2. Cabeçalhos e Parágrafos

Cabeçalhos organizam e estruturam o conteúdo da página, assim como os títulos de um livro. A hierarquia vai de <h1> (mais importante) a <h6> (menos importante).

  • <h1>: Pense como o título da capa de um livro.
  • <h2>: Subtítulos maiores dentro de seções.
  • <h6>: Subtítulos menores para tópicos mais específicos.

Os parágrafos seguem os cabeçalhos, detalhando as informações como o corpo de um capítulo em um livro.

<h1>Cabeçalho 1</h1>
<h2>Cabeçalho 2</h2>
<p>Este é um parágrafo.</p>

3. Links e Imagens

Os links em uma página web funcionam como portas que levam você para diferentes ambientes, ou seja, outras páginas da web. Imagine que você está em uma sala e quer visitar outra; o link é a porta que você usa para se deslocar. Eles conectam sua página a outros lugares na internet de maneira simples e eficaz.


As imagens são como quadros que enfeitam essa sala. Elas não só ilustram o conteúdo como também ajudam a transmitir ideias e emoções sem a necessidade de palavras. Uma imagem bem posicionada pode comunicar uma mensagem de forma muito mais rápida e direta.

Veja na prática:

  • <a href="https://www.google.com">: Este é o link que redireciona o usuário para uma nova página, como o Google.
  • <img src="imagem.jpg" alt="Descrição da Imagem">: Essa tag insere uma imagem na página, com uma descrição fornecida no atributo "alt".

4. Formatação de Texto


Você pode imaginar a formatação de texto como a ênfase que você dá ao falar. Quando você quer destacar algo importante, você pode falar mais alto ou com mais firmeza, assim como o negrito faz no texto. Para ideias mais sutis ou para mostrar que algo é especial, você pode usar um tom mais suave, que é como o itálico no texto.



Vamos testar:


  • <strong>: Essa tag aplica negrito ao texto, destacando-o.
  • <em>: Essa tag aplica itálico ao texto, enfatizando algo de maneira mais suave.

5. Formulários Básicos

Os formulários em uma página web funcionam como questionários que você entrega a alguém para coletar informações. Imagine que você está em uma loja e recebe um formulário para preencher com seus dados — o mesmo acontece online. Os formulários são usados para coletar dados importantes, como nome, email e preferências do usuário.

Veja como:

  • <form>: Esta tag cria o formulário, é como o papel onde as perguntas serão escritas.
  • <input type="text">: Cria um campo onde o usuário pode digitar informações, como seu nome.
  • <input type="email">: Um campo específico para o usuário inserir o endereço de email.
  • <input type="submit">: O botão que o usuário clica para enviar suas respostas.
