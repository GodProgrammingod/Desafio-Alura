Explorando o CSS

Aqui você aprenderá os principais conceitos do CSS e verá interações práticas.

1. Alterando a cor do texto

Com CSS, você pode alterar a cor do texto de maneira simples usando a propriedade color. Vamos ver como funciona:

p {
  color: blue;
}

2. Alterando o tamanho da fonte

Outra funcionalidade importante do CSS é ajustar o tamanho da fonte usando a propriedade font-size.

p {
  font-size: 50px;
}

3. Alterando o fundo de um elemento

Você pode personalizar o fundo de um elemento HTML com a propriedade background-color.

div {
  background-color: yellow;
}

4. Centralizando um elemento

Com a propriedade text-align, você pode centralizar o texto dentro de um elemento.

div {
  text-align: center;
}
